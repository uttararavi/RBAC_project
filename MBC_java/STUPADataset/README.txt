First line of the file specifies the number of users.
The second line , the number of permissions.
The thirs specifies the number of floors.
The third specifies the number of regions/rectangles per floor.
The next line gives the density of population of the matrix.
The next gives the the number of sessions or the time slots availabe.

Each line gives the values in the matrix row wise and at the end of each row there is a blank line to demarkate the start of new row. 

format:

{(floor_number,region_number),start_time,stop_time)}
